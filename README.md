# 💰 Expense Tracker

A full-stack Expense Tracker application built using ASP.NET MVC, Web API (JWT Authentication), Entity Framework, and SQL Server.

---

## 🚀 Features
- 🔐 User Authentication (JWT आधारित login/register)
- 💸 Expense CRUD (Add, Edit, Delete, View)
- 📊 Dashboard with Charts (Chart.js)
- 👤 User Profile Management
- 🔔 Toast Notifications & Loader
- 📅 Date Range Filters
- 🧩 Clean Architecture (UI → API → Service → DB)

---

## 🛠 Tech Stack

- ASP.NET MVC (.NET Framework)
- ASP.NET Web API
- Entity Framework
- SQL Server
- Razor Views
- Chart.js

---

## 📂 Architecture

UI (MVC) → API → Services → Database
